Project Overview

Developing a secure, low-latency handover protocol for 5G networks using quantum-resistant cryptographic algorithms (Kyber/Dilithium) to protect against future quantum computing threats.

Interactive Handover Process

01

Measurement Reporting

User Equipment (UE) sends signal strength reports to the Source gNB.

02

Handover Decision

Source gNB decides to initiate handover based on RSRP/RSRQ thresholds.

03

PQC Key Exchange

Execution of Post-Quantum Cryptographic (PQC) handshake between Target gNB and UE.

04

Execution & Path Switch

UE connects to Target gNB; core network updates the user plane path.

Quantum Resistance

Our solution implements Lattice-based cryptography to ensure that user identities and session keys remain secure even in the "Q-Day" era.
